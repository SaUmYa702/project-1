About the Project

Project Overview: The "Kissan Call Centre Visualization and Analysis" project aims to provide valuable insights into the agricultural concerns of farmers by analyzing data collected from a Kissan (farmer) call centre. The project focuses on visualizing call data such as queries raised by farmers, common issues faced, geographical patterns, and the type of advice provided. Through this project, the goal is to develop a comprehensive understanding of the challenges farmers are facing and how the call centre supports them in resolving these issues.

Purpose

The main purpose of this project is to help stakeholders, such as agricultural experts, government organizations, and NGOs, make data-driven decisions that can enhance support for farmers. By visualizing key data, it aims to improve the efficiency of services provided by Kissan Call Centres and aid in the development of better solutions for the agricultural community.

Objectives

  • To analyze call data and identify common issues faced by farmers.
  • To create visual dashboards that can present trends, patterns, and critical insights from the data.
  • To categorize queries and advice given by the call centre based on geographic regions and crop types.
  • To provide recommendations for improving the services of the call centre based on data analysis.

Scope of the Project:

Data analysis of calls made by farmers to the Kissan Call Centre. Categorization of calls into different types of agricultural issues such as crop diseases, weather-related concerns, pest control, and general farming advice. Visualization of the data using graphs, charts, and heatmaps to identify trends and patterns. Geographic analysis to understand regional farming issues.

Methodology:

Data Collection: The project is based on historical call data from the Kissan Call Centre. Data Cleaning and Preprocessing: Raw data is cleaned to remove inconsistencies and categorized into relevant agricultural issues. Data Analysis: Various statistical and analytical techniques are used to extract meaningful insights. Visualization: Interactive dashboards and visualizations (using tools like Tableau or Power BI) are created to display insights in an easily understandable format.
